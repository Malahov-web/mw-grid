Structure Elements

They are four main standart structure elements: ".section", ".container", ".row" and columuns like ".mv_" or [class*="ds_"] 


<section class="section">
    <div class="container">
        <div class="row">
            <div class="ds_24"></div>
        </div>
    </div>
</section>

Section

Section ".section" is global wrapper on page.

Container

Container ".container" is wrap for rows of columns.

  • ".container" centers your content on page margin-left: auto; margin-right: auto;
  • limits the width of the content. ".container.container-elastic" for no-limit

Row

Row ".row" is wrap for columns

  • Set columns in a row
  • Also adjusts columns gap
  • Control display model: block, inline-block or flex
