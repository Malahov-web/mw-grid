Collapse

Columns can be collapse. Add class ".collapse" to row, container, or single column. It removes columns gaps.

  • Row collapse - collapse all columns in a row, and nested rows
  • Row collapse this - collapse columns in this row only ".collapse-this"
  • Container collapse - collapse container and columns in his child row only
  • Columns Row - collapse only this column
