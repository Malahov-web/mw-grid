Display models

Mw-rid has 3 display models: "block", "inline-block" and "flexbox". And they are CSS-classes for row wrap: ".row", ".row-line" and ".row-flex"

  • 3 models flexbox, block and inline-block
  • Set models in SCSS options ($display_models)
  • Drive models with 3 CSS-classes is deadly simple

Display models is set in SCSS obj $display_models

Block model

Block model is simple and solid oldschool recepsion for make modular grid layout

  • Columns in block row display like blocksfloat: left;
  • Columns order can be reverse (."row-reverse" )

Columns can be nested. Always it's need ".row" wrapper

Inline-block model

Inline-Block is more powerfull than block and more cross-browser than flex. It has vertical aligment columns in row

  • Columns in inline-block row display like inline-blocks display: inline-blocks;
  • Columns order can be reverse (."row-reverse")
  • Columns can be aligned vertically (."middle" and other)
  • Row has nulled font-size (font-size: 0 )

Flex model

Flex is most powerfull way to arrange columns inside rows. It has columns same height and specific columns order.

  • Columns in flex row display like elements in flex container display: flex
  • Columns order can be reverse (."row-reverse")
  • Columns can be aligned vertically (."middle" and other)
  • Columns can be the same height (."stretch")
