SCSS Utilites

Mw-grid SCSS API provides follow functions, which you can use to get some grid parameters. These functions get data from $devices obj.

  • get_grid_bp() - get device breakpoint value
  • get_grid_c_m() - get device container_max value
  • get_grid_f_b() - get device font_base value
  • get_grid_ms_c() - get device ms_coeff (modular scale) coefficient value
