Ordering

For change ordering mw-grid has class ".reverse" (for compatibility, also the old class ".row-reverse") which you can add to row and change columns direction from left-to-right to right-to-left.

  • 1 universal class
  • Use float: right for block model, direction : ltr for inline-block and flex-direction: row-reverse for flex
  • Non-responsive - class set alignment for all breakpoints
  • Supports by all display models
  • !Notice for inline-block model - I forcibly changed the direction for the columns inside the .row-line to default direction : rtl
