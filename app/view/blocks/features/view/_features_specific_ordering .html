Specific order

To set specific columns order we have "order" css-classes, add the key word "order" and number - number of the column in row, like ".ds_order_2"

By default the max number of order number is equal to the number of grid columns.

  • Order number is real element number in CSS
  • Use order
  • Specific order is responsive (".mv_order_3.ds_order_2") up from breakpoints
  • Supports by flex model only
