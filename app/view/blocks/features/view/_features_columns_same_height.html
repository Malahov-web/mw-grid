Columns Same Height

To make columns same height you need just use stretch aligment in ".row-flex". Add class ".stretch" to row.

  • Use align-items: stretch for flex model
  • Non-responsive - class set alignment for all breakpoints
  • Supports by flex model only
  • Notice: stretch alignment in ".flex-row" is set by default
