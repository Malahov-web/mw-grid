Block model

Block model is simple and solid oldschool recepsion for make modular grid layout

  • Columns in block row display like blocksfloat: left;
  • Columns order can be reverse (."row-reverse" )

Columns can be nested. Always it's need ".row" wrapper

"row row-reverse"
column 1
column 2
