Inline-block model

Inline-Block is more powerfull than block and more cross-browser than flex. It has vertical aligment columns in row

  • Columns in inline-block row display like inline-blocks display: inline-blocks;
  • Columns order can be reverse (."row-reverse")
  • Columns can be aligned vertically (."middle" and other)
  • Row has nulled font-size (font-size: 0 )
