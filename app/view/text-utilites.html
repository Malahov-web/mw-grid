Text utilites

Mw-grid has utilites for alignment. You can align text in elements with follow CSS-classes ".lefted", ".righted", ".centered" and ".justifyed"

All alignment values are responsive, add breakpoint prefix to class name like ".tb_centered" to align on tablets and large.

  • 4 values for aligning
  • All values are responsive, have class for each device (".tb_centered")
  • All responsive classes work up from breakpoint
