Columns

Columns in mw-grid has classes like ".mv_" or [class*="ds_"]. Default mw-grid has 5 breakpoints in SCSS object $devices and their keys are CSS-class prefixes:

  • "mv_" - for "mobile vertical" (like ".mv_12") up from 320px
  • "mh_" - for "mobile horizontal" (like ".mh_12") up from 576px
  • "tb_" - for "tablets" (like ".tb_12") up from 768px
  • "mv_" - for "desktops" (like ".ds_12") up from 1024px
  • "hd_" - for "hi definition" (like ".hd_12") up from 1366px

Columns number is set in SCSS var $columns

Columns gaps (left and right) is set in SCSS var $columns_gap
