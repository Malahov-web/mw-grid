Horizontal alignment

You can control the horizontal alignment of columns using offsets.
To form the offset CSS-class to the device name add the key word "offset" and size - the number of columns, like ".ds_offset_2".

  • Use columns offsets
  • Horizontal alignment is responsive - work up from breakpoint
  • Supports by all display models

Vertical alignment

And control the vertical alignment is done using follow classes: ".top", ".middle", ".bottom", and ".baseline"

  • Have 4 universal classes for alignment
  • Use vertical-align for inline-block model, and align-items for flex-model.
  • Non-responsive - class set alignment for all breakpoints
  • Supports by inline-block and flex models
