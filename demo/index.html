Overview

Powerfull and lightweight CSS layout framework for creating your layout like 'container/row/column' support block, inline-block and flexbox models.

SCSS API

You have simple and usable API to set your custom grid parameters.

Var TypeDefaultDescription
Vars
$columns int, unitless 12 columns number
$columns_gap int, unitless 15 column one side gap
$container_width int, unitless 1400 container global max width
$breakpoints_between string,
['responsive' || 'fixed']		 'responsive' behavior between breakpoints
Objects
$display_models map (block: 1, inline-block: 0, flex: 1)) types of display models
$devices map see Expamle devices parameters - breakpoints, container max width, font base, and modular scale coeff for each device

Variables


$columns

columns is global var. Sets the number of columns in grid.


$columns_gap

columns_gap is global var. Sets the padding between columns.

  • Value is column one side gap
  • Use CSS-poperty padding-left, padding-right for containers and columns and margin-left, margin-right for rows
  • Replaced when the 'columns_gap' property is defined on the $device object

Example

In your SCSS style add options and include mw-grid core, like this:

            
// Options: 
$columns: 12;
$columns_gap: 10;
$container_width: 1200;
$breakpoints_between: 'responsive'; // 'responsive' || 'fixed' 
 
$modules_display: ( 
    block: 1,
    inline-block: 0,
    flex: 1
);
 
$devices: ( 
    mv_: (
        breakpoint: 320,
        container_max: 300,
        font_base: 14px,
        ms_coeff: 1.10,
    ),
    mh_: (
        breakpoint: 576,
        container_max: 540,
        font_base: 14px,
        ms_coeff: 1.10,
    ),
    tb_: (
        breakpoint: 768,
        container_max: 750,
        font_base: 15px,
        ms_coeff: 1.17,
    ), 
    ds_: (
        breakpoint: 1024,
        container_max: 960,
        font_base: 16px,
        ms_coeff: 1.17
    ),
    hd_: (
        breakpoint: 1366,
        container_max: 1200,
        font_base: 16px,
        ms_coeff: 1.17
    )   
);  

// Import module
@import 'node_modules/mw-grid/src/_mwgrid.scss';
