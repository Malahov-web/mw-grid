Text utilites

Mw-grid has utilites for alignment. You can align text in elements with follow CSS-classes ".lefted", ".righted", ".centered" and ".justifyed"

All alignment values are responsive, add breakpoint prefix to class name like ".tb_centered" to align on tablets and large.

  • 4 values for aligning
  • All values are responsive, have class for each device (".tb_centered")
  • All responsive classes work up from breakpoint

Responsive classes work up from breakpoins(mobile-first). For expamle with default breakpoints:

Mobile Vertical (mv_)
320px		 Mobile Horizontal (mh_)
576px		 Tablets (tb_)
768px		 Desktops (ds_)
1024px		 High Definition (hd_)
1366px
".centered", ".mv_centered", centered centered centered centered centered
".mh_centered" - centered centered centered centered
".tb_centered" - - centered centered centered
".ds_centered" - - - centered centered
".hd_centered" - - - - centered

.centered
Text is centered on all devices
.mv_centered .td_lefted .hd_righted
Text is righted on HD,
lefted on Tablets,
and centered on Mobiles
